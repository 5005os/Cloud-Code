# 🇰🇬 Кыргызский датасет для AkylAi

Это начало датасета для обучения кыргызскоязычного ИИ. Цель — собрать
качественные данные на кыргызском языке, чтобы в будущем обучить (или
дообучить) собственную модель.

## 📁 Файлы

| Файл | Что внутри | Формат |
|------|------------|--------|
| `translations.json` | Пары перевода русский ⇄ кыргызский (~165) | JSON: `{ "ru": "...", "ky": "..." }` |
| `qa.jsonl` | Пары «инструкция → ответ» для обучения | JSONL (alpaca-style) |
| `proverbs.json` | Кыргызские пословицы с переводом | JSON: `{ "ky": "...", "ru": "..." }` |
| `verbs.json` | Глаголы со спряжением (наст. время) | JSON |
| `dialogues.json` | Диалоги (знакомство, магазин, улица, в гостях) | JSON |
| `numbers.json` | Все числа и правило составных | JSON |
| `texts.json` | Короткие тексты (о КР, Манас, семья) | JSON |
| `articles.json` | Статьи (традиции, кухня, культура, природа) | JSON |
| `constitution.json` | Факты о Конституции КР (источник: shailoo.gov.kg) | JSON |
| `law_qa.jsonl` | Вопросы-ответы по праву и госустройству | JSONL |
| `traffic_rules.jsonl` | ПДД, дорожные знаки, авто-лексика (рус) | JSONL |
| `traffic_rules_ky.jsonl` | ПДД на кыргызском языке | JSONL |
| `pdd_exam.jsonl` | Экзаменационные билеты ПДД с правильными ответами | JSONL |
| `pdd_top50.jsonl` | Топ-50 вопросов-ответов по ПДД КР | JSONL |
| `grammar.md` | Грамматика кыргызского (падежи, гармония, глаголы) | Markdown |
| `phrasebook` (в `data/knowledge.js`) | Разговорник для сайта | JS |

## ➕ Как пополнять датасет

Чем больше **качественных** данных — тем умнее будет модель. Добавляй:

1. **Переводы** — в `translations.json` новые пары `{ "ru": "...", "ky": "..." }`.
2. **Вопрос-ответ** — в `qa.jsonl` строки вида
   `{"instruction": "вопрос", "input": "", "output": "ответ на кыргызском"}`.
3. **Тексты** — кыргызские книги, статьи, новости (с указанием источника).

### Правила качества
- Только **проверенный** кыргызский (без машинного «мусора»).
- Сохраняй кыргызские буквы: **ң, ө, ү**.
- Указывай источник, если берёшь из книги/статьи.

## 🎯 Куда дальше

Когда наберётся достаточно данных (хотя бы тысячи примеров), по ним можно
**дообучить открытую модель** (Llama / Qwen / Gemma) на кыргызский язык.
Пошаговый план — в файле [`TRAINING_PLAN.md`](TRAINING_PLAN.md).

## 📚 Где брать кыргызские тексты (источники)

- Кыргызская Википедия — https://ky.wikipedia.org
- Эпос «Манас» (тексты)
- Кыргызские книги, газеты, новостные сайты
- Учебники кыргызского языка
- Переводы (кыргызско-русские словари)
