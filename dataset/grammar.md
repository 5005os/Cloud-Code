# 📘 Грамматика кыргызского языка (основы)

Кыргызский — тюркский язык. Он **агглютинативный**: к корню слова
«приклеиваются» окончания (как кубики). Здесь — главные правила простыми словами.

---

## 1. Гармония гласных (үндөштүк)

Окончания **подстраиваются** под гласные слова. В кыргызском 8 гласных:

| | твёрдые | мягкие |
|---|---|---|
| неогубленные | а, ы | е, и |
| огубленные | о, у | ө, ү |

Поэтому у одного окончания бывает **4 варианта**. Пример — множественное число
`-лар / -лер / -лор / -лөр`:
- бала → бала**лар** (дети)
- эже → эже**лер** (сёстры)
- тоо → тоо**лор** (горы)
- көл → көл**дөр** (озёра)

> После звонких/глухих согласных вместо `-лар` бывает `-дар` или `-тар`
> (китеп → китеп**тер** — книги).

## 2. Нет рода и артиклей

В кыргызском **нет** мужского/женского рода и **нет** артиклей (как «the/a»).
`ал` = и «он», и «она», и «оно».

## 3. Порядок слов: SOV

Глагол всегда **в конце** предложения:
**Подлежащее → Дополнение → Сказуемое**

- Мен **нан жейм** — Я хлеб ем (= Я ем хлеб).
- Ал **китеп окуйт** — Он книгу читает.

## 4. Местоимения (ат атооч)

| кыргызча | русский |
|---|---|
| мен | я |
| сен | ты |
| сиз | вы (вежливо) |
| ал | он / она |
| биз | мы |
| силер | вы (мн.) |
| алар | они |

## 5. Падежи (жөндөмөлөр) — 6 штук

| Падеж | Вопрос | Окончание (пример с «үй» — дом) |
|---|---|---|
| Атооч (именительный) | кто? что? | үй |
| Илик (родительный) | чей? | үй**дүн** |
| Барыш (дательный) | куда? кому? | үй**гө** |
| Табыш (винительный) | кого? что? | үй**дү** |
| Жатыш (местный) | где? | үй**дө** |
| Чыгыш (исходный) | откуда? | үй**дөн** |

Пример: «Мен үй**гө** барам» — Я иду домой (куда? — в дом).

## 6. Принадлежность (таандык)

«Мой, твой, его» выражается окончанием:

| | пример (китеп — книга) |
|---|---|
| менин (мой) | китеб**им** |
| сенин (твой) | китеб**иң** |
| анын (его) | китеб**и** |
| биздин (наш) | китеб**ибиз** |

## 7. Глагол: настоящее время

Корень + `-а/-е/-о/-ө` + личное окончание:

| | баруу (идти) |
|---|---|
| мен | бар**ам** |
| сен | бар**асың** |
| сиз | бар**асыз** |
| ал | бар**ат** |
| биз | бар**абыз** |
| силер | бар**асыңар** |
| алар | бар**ышат** |

## 8. Отрицание

- Глагол: добавляем `-ба/-бе/-бо/-бө`:
  бар**ам** (иду) → бар**бай**м (не иду).
- Существительное/прилагательное: слово `эмес` (не):
  «Бул китеп **эмес**» — Это не книга.

## 9. Вопрос

Частица `-бы/-би/-бу/-бү` в конце:
- «Сен барасың**бы**?» — Ты идёшь?
- «Бул жакшы**бы**?» — Это хорошо?

---

## 🧩 Кратко

1. Окончания «приклеиваются» к слову и **меняются по гармонии гласных**.
2. **Нет рода**, нет артиклей.
3. **Глагол — в конце** предложения.
4. **6 падежей**, принадлежность — окончанием.
5. Отрицание глагола — `-ба`, вопрос — `-бы`.

> 💡 Главное в кыргызском — почувствовать гармонию гласных. Остальное —
> практика: читай, слушай, повторяй.
